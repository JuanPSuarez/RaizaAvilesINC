## Run Locally

Clone the project

```bash
  git clone https://github.com/JuanPSuarez/raiza-aviles-inc.git
```

Go to the project directory

```bash
  cd raiza-aviles-inc
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
